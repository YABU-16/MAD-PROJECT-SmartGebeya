📱 SmartGebeya

Connecting Buyers and Sellers in One Continuous Marketplace

📖 Project Description

SmartGebeya is is a commercial Android marketplace application designed to modernize local trading by connecting buyers and sellers through a single digital platform. Inspired by Ethiopian market culture (Gebeya), the application enables users to list products, browse available items, and communicate directly within a secure and user-friendly mobile environment.

The project aims to address challenges faced by small merchants and consumers who rely on informal trading methods with limited reach and organization. By providing an accessible mobile marketplace, TradeLoop improves product visibility, simplifies buying and selling processes, and supports local commerce through efficient and transparent digital interactions.
