📱 SmartGebeya

Connecting Buyers and Sellers in One Continuous Marketplace

📖 Project Description

SmartGebeya is a commercial Android marketplace application designed to digitize local trading and connect buyers and sellers through a modern mobile platform. Inspired by Ethiopian market culture (Gebeya), the application provides an accessible and secure environment where users can list products, discover items, and communicate efficiently.

The goal of this project is to solve challenges faced by small merchants and consumers who rely on informal trading methods. SmartGebeya enables wider market reach, improves product discovery, and simplifies transactions through an easy-to-use mobile experience built using modern Android development practices.
